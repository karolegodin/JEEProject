# JEEProject

Pour ouvrir ce projet, lancez-le sur un serveur, et ouvrez-le sur le port correspondant.

Par exemple, sur un serveur local sur le port 8080 :
http://localhost:8080/ProjetJEE

Cela vous amènera à l'accueil du site.

Le site comporte:
- La page d'accueil.

- Une page d'autentification.
   --> Selon les droits du compte, certaines pages ne seront pas accessibles.
   Pour accéder à tous : login "test" et laisser le mot de passe vide.

- Une page listant les joueurs.
   --> Cliquer sur un joueur révèle ses détails.

- Une page listant les matchs.
   --> Cliquer sur un match révèle ses détails.

- Une page de statistiques sur les joueurs.
   --> Cliquer sur une colonne ordonne les listes selon la valeur de la colonne cliquée.

- Une page permettant de modifier les effectifs de joueurs.
   --> Réécrire des champs puis cliquer sur "modifier" modifie le joueur dans la base de données.
   --> Cliquer sur "Supprimer" supprime le joueur correspondant.
   --> Remplir les champs d'ajout puis "Ajouter" crée un nouveau joueur dans la base.
   
- Une page permettant de modifier les effectifs de matchs.
   --> Même fonctionnement que pour les joueurs.
